# 🤖 AND - Adaptive Network Daemon

**AND**는 다양한 통신 환경에 맞춰 최적의 네트워크 모듈을 선택·관리하는 적응형 네트워크 데몬입니다.  
WebRTC, ZeroMQ, ROS2 등 여러 네트워크 프로토콜을 모듈 방식으로 운용할 수 있으며, 원격 로봇 제어나 분산 제어 시스템에서 안정적인 통신을 보장합니다.

---

## ✨ 주요 기능

- 🧠 **Adaptive Switching**: 통신 환경에 따라 자동으로 적절한 프로토콜(WebRTC, ZeroMQ 등) 선택
- 🧩 **Modular Design**: 통신 모듈은 플러그인 방식으로 구성 (간단한 교체 가능)
- 📡 **Daemon Execution**: 백그라운드에서 자동 실행 및 실시간 상태 유지
- 🔒 **Connection Monitoring**: 연결 안정성 검사 및 재시도 로직 내장
